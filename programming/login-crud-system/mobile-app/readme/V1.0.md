📘 Mobile App README v1.0
## 🚀 專案簡介
本 Mobile App 採用 Ionic + Angular + Capacitor 開發，提供登入、產品瀏覽、聯絡頁面及推播通知功能。
前端 UI 使用 Ionic 元件，後端 API 由 CMS 提供，並透過 JWT 驗證 保護資源。

## 🔑 功能特色 (v5.0 更新)
登入頁面：表單驗證，成功後儲存 JWT token。
產品頁面：支援分類篩選 (ion-segment)，產品卡片可點擊進入詳細頁面。
詳細頁面：顯示產品圖片、描述、分類。
聯絡頁面：支援 mailto: 或 API 傳送訊息。
推播通知：整合 FCM，接收後端發送的通知。

## 🔄 資料流流程
使用者登入 → 呼叫 CMS authlogin.php → 成功後取得 JWT。
JWT 儲存於 Capacitor Preferences。
產品頁面 → 呼叫 CMS products.php → 顯示產品列表。
點擊產品 → 跳轉至 /description/:id → 顯示詳細資料。
聯絡頁面 → 可直接寄信或呼叫 API。
後端 sendmessage.php → 透過 FCM 推播至 App。

## 🛠 技術棧
Frontend: Ionic 7, Angular 17, Capacitor
Backend API: PHP CMS (JWT 驗證)
Auth: JWT (HS256)
Push Notification: Firebase Cloud Messaging (FCM)

## ⚡ 如何執行
安裝依賴: npm install
啟動開發伺服器: ionic serve
建立 Android/iOS 平台: ionic capacitor add android/ionic capacitor add ios
同步更新: ionic capacitor sync
