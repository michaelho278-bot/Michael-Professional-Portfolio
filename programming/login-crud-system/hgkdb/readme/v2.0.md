# Token-based API (PHP + PDO + JWT)

## 📖 專案簡介
由傳統 CMS session-based login 升級到 token-based API。
支援前後端分離 (React/Vue/Mobile)，提升安全性同可擴展性。

---

## ⚙️ 架構
- `config.php` → 資料庫連線 + JWT secret
- `auth.php` → Token 生成 / 驗證
- `login.php` → Login API，成功後發 JWT
- `api/user.php` → 範例 CRUD API，需驗證 token

---

## 🔑 Login 流程
1. 前端傳送 `{username, password}` 至 `/login.php`
2. 後端驗證成功 → 發 JWT token
3. 前端保存 token (localStorage / sessionStorage)
4. 每次 API call → `Authorization: Bearer <token>`
